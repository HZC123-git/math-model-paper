# math-model-paper 使用说明

> 数学建模竞赛论文全自动生成 | 数据 → 论文，一条命令

---

## 一、安装

### 方式1：npx 一键安装（推荐）

```bash
npx skills add https://github.com/HZC123-git/math-model-paper.git
```

### 方式2：Git 手动安装

**Windows（PowerShell）：**
```powershell
cd $env:USERPROFILE\.claude\skills
git clone --depth 1 https://github.com/HZC123-git/math-model-paper.git
```

**macOS / Linux：**
```bash
cd ~/.claude/skills
git clone --depth 1 https://github.com/HZC123-git/math-model-paper.git
```

### 验证安装

重启 Claude Code 后，输入：
```
/math-model-paper
```
看到 skill 加载提示即安装成功。

## 二、使用方式

### 最简单：一句话生成论文

```
/math-model-paper
数据: 房价数据.csv
目标列: 房价_万元
标题: 基于多元回归的房价影响因素分析
竞赛: 国赛
```

### 全自动：命令行模式

```bash
cd skills/math-model-paper/pipeline
python pipeline.py 房价数据.csv --target 房价_万元 --type regression --title "房价影响因素分析" --competition 研究生数学建模
```

### 只生成论文（已有分析结果）

```
/math-model-paper
这是一份分析报告：[粘贴JSON数据]
帮我生成研究生数学建模竞赛论文
```

## 三、支持的功能

### 竞赛模板（14种）
说出竞赛名称即可自动匹配模板：
- `美赛` / `国赛` / `研究生数学建模` / `统计建模`
- `MathorCup` / `APMCM` / `电工杯` / `深圳杯`
- `华中杯` / `东三省` / `数维杯` / `五一杯` / `认证杯` / `中青杯`

### 支持的数据格式
- CSV（.csv）
- Excel（.xlsx / .xls）

### 支持的问题类型
- `regression` — 回归/预测
- `classification` — 分类/识别
- `clustering` — 聚类/分组

### 自动完成的事情
- ✅ 12+模型自动对比，选出最优
- ✅ 中英混合文献引用（中文≥5篇）
- ✅ 数学公式自动嵌入（≥5个）
- ✅ 图表自动生成并嵌入
- ✅ 证据校验（数值 vs 数据源对照，不一致不通过）
- ✅ 降重处理（修改率>40%）
- ✅ 去AI味（三层检测）
- ✅ 排版合规检查
- ✅ 魔鬼代言人审校（15项清单）
- ✅ 自评打分（7维加权，≥85分交付）

## 六、常见问题

**Q: 论文太短怎么办？**
A: 确保用的是 skill 调用（`/math-model-paper`），不是直接跑脚本。skill 会逐节 LLM 生成，自动保证字数 8000+。如果还是短，说"帮我把第X节扩展一下"。

**Q: 参考文献全是英文？**
A: 说"补充中文文献"，skill 会加入房价/回归/统计应用等领域的中国学者文献。

**Q: 想改某一章？**
A: 直接说"把 4.3 节改用 XGBoost 重写"或"摘要缩短到 200 字"，不需要全文重跑。

**Q: 论文生成到一半断了？**
A: skill 有断点续传功能，直接调用 `/math-model-paper` 会自动从断点继续。

**Q: 图表颜色不好看？**
A: pipeline 内置了发表级配色方案（色盲友好、学术配色、300dpi）。如需自定义，说"图表用暖色调"即可。
