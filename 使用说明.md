# math-model-paper 使用说明

> 数学建模竞赛论文全自动生成 | 数据 → 论文，一条命令

---

## 快速开始

复制以下提示词，粘贴到 Claude Code 对话框：

```
/math-model-paper

【强制规则】
- 严格按 skill 的 14 步管线执行，一步一确认，每步必须输出可验证的中间产物
- 禁止手工等效替代：不允许"我手动搜了文献所以跳过 literature_search.py"
- 每完成一步，报告：哪个脚本跑了、输出了什么文件、下一步是什么
- 🚫 禁止项不得违反：脚本硬编码论文 / 全英文文献 / 跳过证据校验 / 缺少公式 / 缺少代码附录

【信息】
- 数据：C:/data/housing.csv
- 目标列：price
- 论文标题：基于多元回归的房价影响因素分析
- 竞赛类型：研究生数学建模

【交付标准】
- 每步结束后暂停，等我确认再进入下一步
- 最终：Word + 证据校验报告 + 自评报告，全部 >= 85 分
```

---

| 国际 | 国家级 | 企业/行业 | 区域/特色 |
|------|--------|----------|----------|
| 美赛 MCM/ICM | 国赛 CUMCM | MathorCup | 华中杯 |
| APMCM 亚太赛 | 研究生华为杯 | 电工杯 | 东三省 |
