# 统计建模大赛论文生成器

按照以下流水线生成论文，**每步不可跳过**。

---

## 生成流程

### Step 0: 赛题解析 🆕
同 math-model-paper Step 0。问题类型偏向 regression / causal / evaluation。
- 识别因变量、核心自变量、控制变量
- 判断是否需要 Probit/Logit（二值因变量）、面板模型（如有时间+个体维度）、IV（如用户提供工具变量）

### Step 1: 智能模型池 🆕
同 math-model-paper Step 1，额外强制加入：
- 异方差稳健标准误（HC1/HC2/HC3）
- VIF 多重共线性诊断
- White 异方差检验
- 如果因变量为二值：Probit + Logit
- 如果数据有面板结构：FE + RE + Hausman test
- 12模型对比（线性5 + 树4 + SVM2 + MLP1）

### Step 1.5: 模型反省 🆕
同 math-model-paper Step 1.5。R²<0.5 强制换方法；R²<0.7 建议优化；R²>0.9 放行。

### Step 2: 统计建模模板
固定模板，参数锁定：
- 封面：编号 + 论文题目 + 参赛学校 + 参赛成员 + 指导老师
- 表格和插图清单（在目录后、摘要前）
- 中文摘要（≥400字，含具体系数和显著性）
- 英文 Abstract（≥250 words）
- 目录
- 一、引言（1.1背景 + 1.2文献综述三部分 + 1.3创新点）
- 二、数据说明与变量选择（2.1来源 + 2.2变量表 + 2.3描述统计表 + 2.4相关性矩阵+VIF）
- 三、方法论与研究设计（3.1基准模型方程 + 3.2估计方法 + 3.3内生性讨论 + 3.4稳健性策略）
- 四、实证分析（4.1基准回归表 + 4.2特征重要性 + 4.3稳健性检验表 + 4.4讨论）
- 五、结论与政策建议（5.1结论3-5条附系数 + 5.2建议3-5条 + 5.3不足与展望）
- 参考文献（≥18篇，中文≥6篇，英文≥12篇，GB/T 7714）
- 附录：Python核心代码（≥30行，可独立运行）
- 致谢

### Step 2.5: 文献检索 🔴
同 math-model-paper Step 2.5。中文文献加入：经济研究、经济学季刊、世界经济、地理科学、浙江大学学报等。
英文核心文献：Rosen 1974, Black 1999, Sirmans 2005, Breiman 2001, Friedman 2001, Wooldridge 2019。
**禁止全英文文献**。

### Step 3: 逐节生成（含字数自检 🆕）
🚫 **禁止用 Python 脚本硬编码论文内容。** 必须 LLM 逐节生成。

| 章节 | 最低字数 | 不合格处理 |
|------|---------|-----------|
| 中文摘要 | 400字 | 重写 |
| 英文Abstract | 250 words | 重写 |
| 文献综述（三部分） | 800字 | 重写 |
| 内生性讨论 | 600字 | 重写 |
| 稳健性检验 | 400字 | 重写 |
| 政策建议 | 300字 | 重写 |
| **全文总计** | **8000字** | 逐节补充 |

### Step 4: 组装初排
- 图表编号连续、公式编号连续
- **代码附录 🆕（强制）**：≥30行 Python 代码（数据加载 + 模型定义 + 训练 + 评估），Consolas 10pt
- **没有代码附录 = 不合格**

### Step 4.5: 证据校验 🔴
同 math-model-paper Step 4.5。论文数值 vs JSON 逐项对比。
**不通过 = 全部暂停。**

### Step 5: 降重处理
同 math-model-paper Step 5。修改率>40%。图表/表格/标题/参考文献 protected 跳过。

### Step 6: 去AI味终扫
同 math-model-paper Step 6。额外检查：统计论文特有 AI 味——
- "具有重要的理论意义和现实意义" → 具体说
- "效果良好" → 报告 R²/p值
- "模型通过了检验" → 说清楚什么检验、统计量、p值

## 写作风格速查

### 正确结果报告
"表4第(1)列报告了基准OLS回归结果。核心解释变量area的系数为0.703(p<0.01)，表明在控制其他特征后，面积每增加一个标准差(19.625m²)，房价提高0.703个标准差(约50.1万元)。"

### 必须报告的统计量
每个回归系数旁标注标准误（括号内）和显著性星号（***/**/*）
每张表格下方标注：N, Adj-R², F-statistic
异方差检验：White test χ²(p值)
多重共线性：Mean VIF, Max VIF

### 绝对禁止
"随着...的发展" | "具有重要意义" | "效果良好" | "模型通过了检验" | 全英文参考文献 | 🚫Python脚本硬编码论文

## Few-Shot 示例库（来自真实获奖论文）

### 摘要（2019碳排放EKC）
"本文主要建立非参数面板数据优化模型，验证环境库兹涅茨曲线的分支中国碳排放量的库兹涅茨曲线是否成'倒U型'分布。采用2006—2017年的省际碳排放量的面板数据和人均GDP面板数据，首先进行Poolability检验确定最佳模型，其次利用交叉检验法和创新算法相结合确定最佳窗宽，接着用一阶差分、组内变换消除固定效应，最后运用局部线性估计进行模型求解。我们发现运用非参数面板数据优化模型给出的EKC曲线，不再是'倒U型'的，而是呈现为'N型'，这说明环境不会随着经济增长而自发改善。"

### 文献综述（2019碳排放EKC）
"通过查看已有文献发现1991年提出的环境库兹涅茨曲线，可以用来描述碳排放量与经济发展水平之间的'倒U型'关系。关于EKC曲线的参数估计研究完善，但是非参数面板数据估计验证上文献很少。"

### 内生性讨论
"OLS估计面临两个潜在的内生性问题。首先是遗漏变量偏误——能力、家庭背景等不可观测因素同时影响受教育年限和工资水平，导致OLS高估教育回报。其次是测量误差——自报的受教育年限可能存在回忆偏差，导致衰减偏误。本文采用工具变量法处理内生性：选取'兄弟姐妹数量'作为受教育年限的工具变量，理由是..."

### 稳健性检验
"本文从三个维度进行稳健性检验：第一，替换因变量——将对数房价作为因变量重新估计，核心系数符号和显著性未发生改变。第二，缩尾处理——对所有连续变量在1%和99%分位进行Winsorize处理后重新估计。第三，Bootstrap随机剔除——随机剔除10%样本重复100次，报告系数的均值和标准差。"
