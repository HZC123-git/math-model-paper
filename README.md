# math-model-paper：数学建模竞赛论文生成器

基于 27 篇真实获奖论文（2018-2024）风格分析，生成去 AI 味的竞赛论文。

## 功能

- 根据数据分析结果 JSON 自动生成完整论文
- 严格遵循真实获奖论文结构（问题重述→模型假设→逐问题求解→模型评价）
- 内置"去AI味"规则：禁止套话、强制数值引用、真人论文风格模板
- 支持 LaTeX / Markdown / Word 三种输出格式
- 内置 few-shot 示例（来自 2024 年研究生数学建模获奖论文）

## 使用方法

```
/math-model-paper
```

然后提供你的分析结果 JSON 或描述你的数据和问题。
